jQuery.kabuki
=============
example showcase, [Github Page](http://esehara.github.com/jquery-kabuki/).

What's jQuery Plugin Kabuki ?
-----------------------------

'Kabuki' is Japanese Traditional Type and Art.

Get Start
---------

* initialize element bind "Kabuki".

    $('#foobar').kabuki().rainbow();

Method 
------

* earthquake()
* rainbow()
* bownd()
